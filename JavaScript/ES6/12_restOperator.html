ES6 Rest Operator

  1. rest Operator allows us easily to handle various input as parameters in a function.

  2. Its syntax allows a function to treat indefinite number of arguments as an array.

  3. ...rest Operator will be last in Parameter () and it will print as an Array.

  4. We can pass many values in Arguments and it will recevied by ...Rest in Parameter() and merge it with other values and create a new function every time for re use.
