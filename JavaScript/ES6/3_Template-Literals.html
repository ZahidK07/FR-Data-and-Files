Template Literals

  1. Template Literals was introduced in ES6 2015.

  2. It helps to call the Strings in JS and other elements to fetch easily and show dynamically with the help of a Template.

  3. It's used with bacticks which is under escape key and Dollar sign with curly braces `${myName}` and target the variable which holds are string to show.

  4. We can also use functions() in Template Literals `${myFunc()}`.

  5. Temlates can also take numerical variables to add substract and calculation purpose.
