  1. In Obeject Oriented Inheritance make easy and helps to create a new class as similar to the Parent and can share those objects to child class we can add some new objects and values in child class to make it different.

  2. To Inherit Objects from Parent class to child class we use extends keyword after child class name.

  3. If we are expecting 2 pramaters (_,_) in constructor then we have to declare 2 arguments when we create a new Object variable otherwise it will Sytax Error.

  4. super() method is mandatory to be written in Child class constructor to inherite the objets from Parent.
    Same parent obejcts are required to call in constructor() parameter to be use by new Arguments.
