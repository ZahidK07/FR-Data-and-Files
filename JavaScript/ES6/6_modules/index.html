Export Import Modules ES6

  1. Import and Export introduced in JS ES6 to manage the code from multiple files which divided and to recieve in main file

  2. We can create Modules in JS and this concept used in ReactJs

  3. What we can Export
    • A variable
    • An object literal
    • A Class
    • A function
    • A ++

  4. To export a code from a file to main file we need to use export before a variable.
    • export let myData = 'Data';

    Export a function(); from file.js
    • let myFunc = ()=>{}

    Importing a func() in main.js
    • import {myFunc};
      • print directly myFunc(); in the main file to get the output

  5. If we're importing 2 variables from same file we can do
    import {myVar, newVar} from 'file.js';
