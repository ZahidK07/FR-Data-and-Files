ES6 Spread Operator


  1. ...spread Operator also starts with three dots to iterate over Arrays.

  2. it will return the Array Data with all together and it can be store in new variable to add and merge with another Array Data to make it one Array.

    let fruitArr = ['Apple', 'Banana','Chicku'];
    let dryfruitArr = ['Almonds','Kaju','Dates'];
    let stockArr = [...fruitArr, ...dryfruitArr];
    console.log(stockArr);
    >>['Apple', 'Banana', 'Chicku', 'Almonds', 'Kaju', 'Dates']
