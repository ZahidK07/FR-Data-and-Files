ES6 Ecma Script:

ES6 is known as Ecma Script which is Standardised by European Computer Association (ECMA) Script.

Provide Standard to JavaScript for different Browsers.

Also used with Node.js & React.js .

This is not a Framework or Library .

ES6 was released in 2015 also called as ES2015.


Variables var, let and Const
  1. let and const do not allow hoisting in ES6
