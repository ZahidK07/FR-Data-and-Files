ES 6 Arrow Functions

  1. functions holds parameteres to get the new values everytime we call the same func to reuse it.

  2. Arrow functions are simple and holds the concept of this keyword.

  3. It uses its Parents this from Window because arrow function doesn't have its own this.

  4. If we required new output to print we have to use new Arguments whithin a function(Argument,Argument) when called

  5. ES6 functions will create with const or let like a variable for example: let myFunc = ()=>{ code block to Print};

  6. arrow functions Parameters can also be used with Template Literals and even the func() can be called in a Template Litral.
