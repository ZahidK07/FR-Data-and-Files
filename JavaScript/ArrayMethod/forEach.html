forEach Method

  1. forEach() is introduced in ES6 to iterate over Arrays and loop over it.

  2. here is the Syntax array.forEach( callback, thisObject )

    1. callback: This allow the function to test the each and every element in the array.
    2. thisObject: This will be called when the callback function is executed.

  3. We can also Iterate over API Array of Objects witht he help this method

  4. The forEach() method contains a call back function and it must take atleast on parameter which represents the elements of an array.
