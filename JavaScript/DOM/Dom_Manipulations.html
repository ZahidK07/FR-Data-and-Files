DOM Manipulation

  1. .innerHTML is an HTML property in DOM to change the the text of an Element.

  2. .style.color = '' or .style.background is used to set Element CSS, we can chane the style by calling the CSS .style attribute and give the CSS property to change color, background, font etc.

  3. .classList.add() use to add a new class to an Element.

  4. .classList.remove() use to remove a class from an Element.
