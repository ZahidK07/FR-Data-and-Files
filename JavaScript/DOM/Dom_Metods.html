DOM Methods

  1. DOM is known as Document Object Model and to Target HTML through JS.

  2. HTML Elements called as Node in DOM and we can manipulate and make changes in it with DOM.

  3. To find HTML with DOM Methods are document.getElementByTagName, ById, ByClassName.

  4. getElementsByTagName('li') It will give all the li Tags HTML Collection.

  5. getElementsByClassName('list'); it will give all the Element Collections of class 'list'.

  6. querySelector is like a CSS selector class Element can be select by .class and Id by #Id.

  7. querySelectorAll('h2'); gives the Node List of Elements selected.
